Трубный - труба, закреплённая с одного конца к внутреннему стакану металлорукава, применяется при транспортировке среды с высокой температурой (например выхлопные газы), для предотвращения прогорания сильфона. Применяется на изделиях небольшой длины, так как трубный внутренний экран не позволяет рукаву работать на изгиб.

Вальцованный - негерметичная несущая оболочка из нержавеющей стали, образующейся путем профилирования и свертки по спирали. Используется в качестве внутреннего экрана для усиления конструкции, рассеивания температуры рабочей среды для предотвращения повреждения основного сильфона.

Оплеточный - стальная сетчатая оплетка, помещенная внутрь сильфона, устанавливается исключительно для рассеивания температуры рабочей среды для предотвращения повреждения основного сильфона.

PTFE - фторопластовая трубка, применяется для исключения контакта высоко агрессивных сред с материалом сильфона, концевой арматуры, как правило используется в химическом производстве, работе с кислотами, химическими соединениями либо высокоадгезийными средами.

Внутренний экран

защитный элемент рукава, помещенный внутрь гибкой части позволяющий исключить/сократить контакт рукава со средой, либо за счет своего наличия продлить срок службы изделия.
